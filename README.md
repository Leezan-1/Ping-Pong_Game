# Ping-Pong_Game
Python programmed  Ping-Pong game

# How to play game
- Install Python 3.0 or above
- Run main.py

# Instructions
For Right player:
- 'Up' key to get right paddle up
- 'Down' key to get right paddle down

For Left Player:
- 'w' key to get left paddle up
- 's' key to get left paddle down
